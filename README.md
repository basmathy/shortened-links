# shortened-links
Проект представяляет из себя сервис коротких ссылок, работающий локально на компьютере. При выходе из приложения через команду `quit` все данные сохраняются в файл `data.txt`, при входе читаются из файла и заносятся в память программы.

## Используемые команды
`signin` - авторизация пользователя по UUID. После ввода команды необходимо указать сам UUID.
`reduce` - сокращение ссылки. После ввода команды необходимо ввести ссылку для сокращения. Если пользователь не авторизован, то одновременно происходит авторизация пользователя. Сокращенная и изначальная ссылка сохраняются в памяти пользователя.
`edit` - используется для редактирования (удаления) текущих ссылок пользователя (если он авторизован). Выводит только ссылки текущего пользователя и позволяет удалять только его же ссылки.
`use` - перейти по короткой ссылке. После ввода команды надо ввести короткую ссылку. Позволяет переходить по ссылкам только авторизованным пользователям и только по их собственным ссылкам.
`signout` - выход из состояния авторизованности.
`quit` - выход из программы.

## Запуск проекта
```
mvn -q package
java -jar target/shortened-links-1.0.0.jar
```
